Generating the fronend:-
The frontend is ReactJS with Tailwind CSS

Building the frontend:-
1.  Initialize your repo with NPM to run vite
2.  npm create vite@latest . -- --template react
3.  Install dependencies: npm install
4.  Move your landing_page.jsx back: mv ../temp_landing.jsx landing_page.jsx.
5.  Replace the content of src/App.jsx with the code from landing_page.jsx 
    (or import it as the main component).
6.  npm install -D tailwindcss postcss autoprefixer
7.  Manually create the tailwind.config.js
8.  Manually reate postcss.config.js in the frontend
9.  Update src/index.css
10. Run: npm install -D @tailwindcss/postcss.
11. Update postcss.config.js
export default {
  plugins: {
    '@tailwindcss/postcss': {},
    autoprefixer: {},
  },
}

6. Start the dev server: npm run dev
7. The app will run on http://localhost:5173 by default.
8. In the API gateway (Orchestrator service) set up the @CrossOrigin
   to accept requests from http://localhost:5173
9. Commit:
    package-lock.json- Locks the vresions of dependencies used by fronent project
    commit to version control for reproducible results.
